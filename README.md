# Nokia (nokia-corp)

Nokia (Nokia Oyj) is a Finnish multinational telecommunications, information technology, and consumer electronics corporation headquartered in Espoo, Finland. Nokia is a leading vendor of carrier-grade network equipment for communication service providers (CSPs), webscalers, and enterprises across mobile (5G/6G RAN, Core), fixed (broadband, fiber), IP (Service Router OS / SR Linux), optical (WaveSuite, 1830 PSS/PSI), submarine cable, and cloud and network services.

## APIs

### Nokia Network as Code Platform
CAMARA-compliant, production-ready Network APIs exposing 5G core and RAN functions through standardized northbound interfaces. Includes Quality on Demand, Device Location (Verification and Retrieval), Geofencing, Specialized Networks (Slicing), Device Status (Roaming, Reachability), Congestion Insights, SIM Swap, Number Verification, Call Forwarding Signal, Device Swap, and KYC (Match, Age Verification, Tenure, Fill-in). Aligned to CAMARA, TM Forum, and GSMA Open Gateway. Available on Google Cloud Marketplace.

**Human URL:** [https://networkascode.nokia.io/](https://networkascode.nokia.io/)

### Nokia Network Services Platform (NSP) API
REST-based automation and assurance API for Nokia's multilayer (IP + optical) Network Services Platform. Provides programmable model-driven service orchestration, inventory, fault, performance, and intent-based automation across multivendor networks.

**Human URL:** [https://network.developer.nokia.com/](https://network.developer.nokia.com/)

### Nokia SR OS pySROS SDK and YANG Models
pySROS is the official Python SDK for model-driven configuration, operational data retrieval, and automation against Nokia SR OS devices (7x50 SR/IXR/XRS/MG and family). Backed by the public 7x50_YangModels repository providing canonical YANG models for NETCONF / RESTCONF / gNMI consumption.

**Human URL:** [https://github.com/nokia/pysros](https://github.com/nokia/pysros)

### Nokia SR Linux
Modern, Linux-based, fully model-driven network operating system for data-center and service-provider routing. Exposes gNMI / gNOI and NETCONF interfaces with YANG as the canonical configuration and state contract.

**Human URL:** [https://www.nokia.com/ip-networks/service-router-linux-NOS/](https://www.nokia.com/ip-networks/service-router-linux-NOS/)

### Nokia WaveSuite Optical REST APIs
Optical networking REST APIs (Swagger-documented) across the WaveSuite portfolio — WaveSuite-NOC, WaveSuite Resource Control, WaveSuite Optimizer, Planner, Service Enablement, and Health & Analytics — plus the NE Management Interface for the 1830 optical platforms.

**Human URL:** [https://network.developer.nokia.com/](https://network.developer.nokia.com/)

### Nokia Network Exposure Platform
Operator-facing platform that enables CSPs to expose their own network capabilities as standardized CAMARA / GSMA Open Gateway APIs to developers and aggregators. Sits beneath Network as Code and provides API lifecycle, monetization, and consent management for service provider API programs.

**Human URL:** [https://www.nokia.com/programmable-networks/network-as-code/](https://www.nokia.com/programmable-networks/network-as-code/)
